# 📊 Retail Sales Analysis Project

## 🧾 Project Title

Sales Uplift: Strategy Insights from Multi-Region Retail Data

---

## 👨‍💻 Objective

The objective of this project is to analyze retail sales data using SQL Server and visualize key business insights using Power BI.

---

## 🛠️ Tools Used

* SQL Server (SSMS)
* Power BI Desktop

---

## 📂 Dataset Description

The dataset contains the following fields:

* TransactionID
* Date
* ProductName
* Category
* Region
* SalesChannel
* Quantity
* UnitPrice
* TotalAmount
* PaymentMode
* CustomerID

---

## 🧩 Project Steps

### 1. SQL Analysis

* Created database and table in SQL Server
* Inserted retail transaction data
* Performed analysis using queries:

  * Region-wise sales
  * Monthly sales trend
  * Top products
  * Online vs Offline comparison
  * Repeat customers

---

## 📊 Key Insights

1. Customers C001 and C002 are the most valuable customers.
2. Necklace and Ring are the top-performing products.
3. Online sales are higher than offline sales.
4. South region has the lowest sales and needs improvement.
5. Monthly sales show peak performance in May.
6. Jewelry category shows consistent demand.
7. Repeat customers contribute significantly to overall revenue.
